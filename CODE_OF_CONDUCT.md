# Code of Conduct

This project adopts the **[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)**
as its code of conduct. The full text is available at that link.

## In short
- Be respectful, collaborative, and constructive.
- Assume good intent; give and accept feedback gracefully.
- Unacceptable behaviour (harassment, personal attacks, or discrimination of any kind)
  will not be tolerated.

## Scope
Applies within all project spaces (issues, pull requests, discussions) and when an
individual represents the project in public spaces.

## Reporting
Report concerns privately to the repository owner via GitHub — **[@aydogandagidir](https://github.com/aydogandagidir)**
(open a private [security advisory](https://github.com/aydogandagidir/conveyor-sorting-twin/security/advisories/new)
if you prefer a confidential channel). Reports are handled discreetly and in good faith,
following the Contributor Covenant enforcement guidelines.
